# Sprites do Jogo

Coloque aqui os PNGs dos personagens e armas.

## Especificações

| Tipo | Tamanho no jogo | Tamanho pra desenhar | Formato |
|---|---|---|---|
| Personagem | 40×60 | **160×240** | PNG transparente |
| Arma pequena | 30×20 | **120×80** | PNG transparente |
| Arma grande | 60×20 | **240×80** | PNG transparente |

## Regras

- Fundo **transparente** (não branco)
- Personagem virado pra **direita** (código espelha)
- Pixel art recomendado — mais fácil e estilizado

## Nomes dos arquivos

- `personagem1_idle.png` — parado, olhando pra direita
- `personagem2_idle.png`
- `arma_pistola.png`, `arma_espingarda.png`, `arma_bastao.png` (ou o que você preferir)
